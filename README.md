# Tibeuru CLI — releases

Public download for `tibeuru.exe`.

## Install

**One-liner (recommended):**

```powershell
irm https://studio.tibeuru.com/install.ps1 | iex
```

**Manual:** [grab the latest `tibeuru.exe`](https://github.com/Tibeuru/tibeuru-cli-releases/releases/latest) and put it on your PATH.
